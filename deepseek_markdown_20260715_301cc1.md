# 🤝 Telegram Dating Bot with Mini App

Бот для анонимных знакомств с Telegram Mini App интерфейсом.

## ✨ Особенности

- Анонимное общение без фото и личных данных
- Telegram Mini App для удобного интерфейса
- Случайный подбор собеседников
- Возможность менять собеседников
- Работает на Python с библиотекой python-telegram-bot
